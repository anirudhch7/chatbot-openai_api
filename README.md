1| **Chatbot using OpenAI's GPT-3.5 and Flask**
2| 
3| This is a simple web application that demonstrates how to create a chatbot using OpenAI's GPT-3.5 model and Flask framework.

4| ## Features
5| - User-friendly web interface
6| - Integration with OpenAI GPT-3.5 for generating chatbot responses
7| - Built with lightweight Flask framework for simplicity

8| ## Prerequisites
9| 
10| Before running the application, make sure you have the following installed:
11| 
12| - Python (>= 3.6)
13| - Flask (pip install Flask)
14| - OpenAI Python library (pip install openai)
15| 
16| ## Setup: Getting Started
17| 
18| - First, set up your OpenAI API key:
19|   - Sign up for an account on OpenAI.
20|   - Replace openai.api_key = "" in app.py with your actual API key.
21| 
22| - Run the application on your localhost:
23|   ```bash
24|   python main.py
25|   ```
